HTML HTML 5
In older Html external plugins like Flash required for for using multimedia on web pages. In Html 5 we don't need any plugins for multimedia support. Html 5 introduced audio and video tags which directly allows to add multimedia on web pages.
In older Html to specify the structure of web pages by using <div> tags and different class and ids. Html 5 introduced <header> <footer> <nav> <main> <article> tags which gives proper stricture to web page and also give it a meaning.
Older version of Html had limited input tags and requires additional javascript to give more functionality. Html 5 introduced more inputs tags like email,tel,url,date etc. for user input.In Html 5 no additional scripting required for additional functionality like validation in form.
